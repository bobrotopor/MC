Окно исходного кода

Окно исходного кода - это главное окно в любой отладочной работе. Оно создаётся когда открыт объектный файл и присутсвует в каждой работе. Если закрыть окно исходного текста, отладочная работа будет завершена.

Marks
Этот указатель(желтая стрелка) показывает расположение программного счетчика и другие отладочные точки (точки останова и trace control statements). Точки останова легко устанавливаются и убираются в окне исходного кода.

Debug Modes (отладочные режимы)
Исходный код может быть показан в нескольких удобных режимах.

Browse the modules (переключение между несколькими модулями)
Объектный файл может состоять из нескольких модулей. Только один модуль может быть показан, но пользователь может выбрать интересующий его модуль в этом меню.

Copy text to Windows Clipboard (копирование текста в Windows Clipboard)
Окно исходного кода поддерживает Windows Clipboard. Пользователь может выбрать часть (или весь) текст находящийся в окне исходного кода и скопировать его при помощи Windows Clipboard выбрав COPY в меню EDIT.

Search (Поиск)
Окно исходного кода поддерживает свободный поиск текста в данном или всех модулях. Первое найденое значение выделяется.

Find Next (Искать далее)
Нажмите искать далее (Нажмите кнопку или клавиши Ctrl-N), чтобы найти следующее значение.
