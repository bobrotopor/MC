Точки остановки
Пользователь может устанавливать безграничное количество точек остановки.

Установка точек остановки
Эта команда устанавливает точку остановки там где в данный момент находится курсор. Эта функция работает только при активном окне исходного кода.

Убрать все точки остановки
Эта функция убирает все установленные точки остановки.

Показать список
Когда выбран пункт показать список, появляется диалоговое окно на экране. В этом окне, пользователь может проверить установленные точки остановки, добавить новые, удалить ненужные или включить/выключить точки остановки.
