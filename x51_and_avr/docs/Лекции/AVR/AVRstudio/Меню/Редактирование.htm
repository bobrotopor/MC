Меню редактирования
Copy (Копирование) (Ctrl-C)
Окно исходного кода поддерживает Windows Clipboard. Пользователь может выбрать часть (или весь) текст находящийся в окне исходного кода и скопировать его при помощи Windows Clipboard выбрав COPY в меню EDIT.

Find (Поиск) (Ctrl-S)
Окно исходного кода поддерживает свободный поиск текста в данном или всех модулях. Первое найденое значение выделяется.

Find Next (Искать далее)
Нажмите искать далее (Нажмите кнопку или клавиши Ctrl-N), чтобы найти следующее значение.
